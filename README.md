# Swift OOP Labs

## Overview

This repository contains my Swift implementations of Object-Oriented Programming and Data Structures labs originally assigned in an Advanced OOP course.

Instead of completing the labs in Java, I am reimplementing them in Swift as part of my journey toward iOS development. The goal is not only to solve the problems but also to understand how core computer science concepts translate into Swift and modern software engineering practices.

Each lab focuses on a specific topic such as graph traversal, stacks, queues, immutability, protocols, generics, error handling, and object-oriented design.

---

## Why This Repository Exists

During my software engineering studies, I learned many important concepts through Java-based coursework.

As I started learning Swift for iOS development, I realized that simply learning Swift syntax was not enough. To truly understand the language, I needed to rebuild familiar concepts using Swift and compare the design patterns, language features, and engineering approaches used in both ecosystems.

This repository serves as:

* A structured Swift learning journal
* A collection of data structure and algorithm implementations
* A place to practice object-oriented design in Swift
* A reference for comparing Java and Swift concepts
* A record of my hands-on learning and experimentation
